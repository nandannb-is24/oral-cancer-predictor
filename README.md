Oral Cancer Risk Predictor
AI-powered image analysis + clinical risk assessment

This project is a Streamlit-based web application that predicts oral cancer risk using:

A deep learning model (CNN) trained on intra-oral lesion images

A clinical questionnaire (smoking, alcohol, prior lesions, symptoms, etc.)

A combined risk scoring system that merges image-based and patient-based factors

The goal is to create a simple, fast, and effective screening system that helps with early detection.
